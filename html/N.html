Rules
x and y stand for a numeral variable signs
k,m,n stand for terms
f,g,h stand for function signs

     f(x) ~ g(x)
     -----------
     f(n) ~ g(n)

        m ~ n
     -----------
     f(m) ~ f(n)

        k ~ m
        k ~ n
        -----
        m ~ n

   f(1+x) ~ x g f(x)
   -----------------
     f(x) ~ g^x f(0)

where

     f(n) ~ (f n)
  (m)f(n) ~ (m f n)

  x(f g)y ~ f x g y

x(f g h)y ~ (x f y) g x h y


  (g^0 n) ~ n
(g^1+x n) ~ x g g^x n

      x+0 ~ x
    x+1+y ~ 1+x+y

      0-1 ~ 0
  (1+x)-1 ~ x

      x-0 ~ x
    x-1+y ~ (x-y)-1

      x*0 ~ 0
    x*1+y ~ x+(x*y)
