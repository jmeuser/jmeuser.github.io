Arithmetic Operators
+  plus
-  minus
*  times
/  quotient
%  remainder|residue (same sign as divisor: 3%(-6) gives -3)
** power
// floor division

Comparison Operators
== equal
!= not equal
<> not equal
>  more
<  less
>= more or equal
<= less or equal

Assignment
=   gets
+=  plus gets (x+=y abbreviates x = x + y)
-=  minus gets
*=  times gets
/=  quotient gets
%=  residue gets
**= power gets
//= floor quotient gets
