Arithmetic Operators
+  plus
-  minus
*  times
/  quotient
%  remainder|residue (same sign as divisor: 3%(-6) gives -3)
** power
// floor division

Comparison Operators
== equal
!= not equal
<> not equal
>  more
<  less
>= more or equal
<= less or equal

Assignment
=   gets
+=  plus gets (x+=y abbreviates x = x + y)
-=  minus gets
*=  times gets
/=  quotient gets
%=  residue gets
**= power gets
//= floor quotient gets

Strings
'...'  string
"..."  string
'\''   single quote
"'"    single quote
'"'    double quote
"\""   double quote
escaped characters are shown in REPL, but not from print(x)
r'...' raw string i.e. escape characters are not interpreted

"""...
......
...""" multi-line string (contains input newlines)

'''...
......
...''' multi-line string (contains input newlines)

"""\
...\
..\""" anew line immediately after \ is ignored

"c" 'c' adjacent string constants are automatically concatenated
