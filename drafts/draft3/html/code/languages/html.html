HyperText Markup Language (HTML)

HTML is punctuation. For example, <q>Hi orb.</q> is code for Hi orb.

Bare Page


<html>
HTML text
</html>
      

<!DOCTYPE html>
<html>
<head>
 <!-- comment text -->
 <meta charset="utf-8">
 <title>title text</title>
</head>
<body>
 <h1>header text</h1>
 <p>paragraph text</p>
</body>
</html>

Preformatted Code

This is idiomatic HTML for punctuating text as preformatted code: 


<pre><code>
preformatted code text
</code></pre>
