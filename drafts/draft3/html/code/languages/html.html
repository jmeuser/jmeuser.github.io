Hypertext Markup Language (HTML)

HTML tells computers the parts played by text in a document.

The Fundamental Example

<!DOCTYPE html> <html> <head> <!-- This text plays the part of a comment. --> <meta charset="utf-8"> <title>This text plays the part of a title.</title> </head> <body> <h1>This text plays the part of a header of the first degree.</h1> <p>This text plays the part of a paragraph.</p> </body> </html>
