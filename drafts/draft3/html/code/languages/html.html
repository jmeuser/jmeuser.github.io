HyperText Markup Language (HTML)

HTML shortens the names for the roles played by text in a document.

Basic Form


<!DOCTYPE html>
<html>
<head>
 <!-- This text is a comment. -->
 <meta charset="utf-8">
 <title>This text is a title.</title>
</head>
<body>
 <h1>This text is a header.</h1>
 <p>This text is a paragraph.</p>
</body>
</html>
