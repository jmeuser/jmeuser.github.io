The Canvas

As in art, SVG graphics appear on a canvas.

Example

The perimeter of a rectangle is shown on this 300px by 100px canvas.


<svg
 width="300px"
 height="100px"
>
 <rect
  x="0px"
  y="0px"
  width="300px"
  height="100px"
  fill="transparent" 
  stroke="black" 
  stroke-width="1px"
 />
</svg>

Example

This canvas has a width of 300px and a height of 100px. The perimeter of a rectangle, this time with a width and height 2px less than those of the canvas, is drawn in black.


<svg
 width="300px"
 height="100px"
>
 <rect
  x="1px"
  y="1px"
  width="298px"
  height="98px"
  fill="transparent" 
  stroke="black" 
  stroke-width="1px"
 />
</svg>
